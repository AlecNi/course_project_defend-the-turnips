
# 保卫萝卜

基于Cocos2d-x平台的保卫萝卜游戏


## 在本地运行

安装并建立Cocos2d-x项目
教程如下
```bash
  https://blog.csdn.net/Gosick_Geass_Gate/article/details/109336995   

```
Clone 这个 project

```bash
  git clone https://github.com/AlecNi/course_project_defend-the-turnips
```

用git下来的Classes和Resources文件夹替换刚刚建立的Cocos项目中的这两个文件夹
